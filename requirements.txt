Online Movie Ticket Reservation System - Requirements Manifest

This is a text manifest, not an executable installer. For one-command setup on Windows, run `setup.ps1`.

Required software:
- Java Development Kit (JDK) 8 or newer
- Apache Maven 3.8 or newer
- Apache Tomcat 9.0.x
- A modern web browser

Project-specific notes:
- The app uses `javax.servlet` APIs, so Tomcat 9 is recommended.
- Build output is a WAR file generated by Maven: `target/OnlineMovieTicketSystem.war`.
- The application uses file-based text storage, so keep the included `.txt` data files with the project.

Build command:
```bash
mvn clean package
```

Deploy the generated WAR to Tomcat 9 and open:
http://localhost:8080/OnlineMovieTicketSystem/

For automated setup, use:
```powershell
Set-ExecutionPolicy -Scope Process Bypass
.\setup.ps1
```

Default demo accounts:
- Admin: admin / admin123
- Customer: customer1 / pass123
